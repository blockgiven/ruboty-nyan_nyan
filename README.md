# Ruboty::NyanNyan

ruboty plugin for say nyan nyan =^･ω･^=

![screenshot](screenshot.png)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-nyan_nyan'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-nyan_nyan

## Usage

    @ruboty nyan nyan
    @ruboty にゃんにゃん
